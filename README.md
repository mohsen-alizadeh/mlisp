* operations accepts exactly two parameters. 
* ignores new lines.
* supports operators : + - * /
* after parsing each expression, prints the output
* supports nested expressions
* single line comments ';'

example:

```
(+ 5 10)
(- 10 2)
(* 5 5) (- 10 5)
(+ (+ 2 4) (+ 1 1))
; comments
```

output:
```
15
8
25
5
8
```
